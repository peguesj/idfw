# Keyboard Shortcuts

IDFWU's most common operations are bound to keyboard shortcuts for speed. These work anywhere in the main window unless a text field has focus.

## Global Shortcuts

| Shortcut | Action |
| --- | --- |
| `Cmd+N` | Add Project |
| `Shift+Cmd+R` | Sync All |
| `Cmd+E` | Toggle Event Stream |
| `Cmd+,` | Preferences |
| `Cmd+?` | Help |

## Notes

- **Add Project** (`Cmd+N`) opens the connector setup sheet. You can pick a local folder, a GitHub repo, or any other registered connector.
- **Sync All** (`Shift+Cmd+R`) re-runs discovery across every enabled scan root and refreshes remote connectors. Use this after pulling new work from multiple machines.
- **Toggle Event Stream** (`Cmd+E`) shows or hides the live Event Stream inspector panel without losing your current selection.
- **Preferences** (`Cmd+,`) opens the standard macOS settings window for scan roots, connectors, APM, and advanced options.
- **Help** (`Cmd+?`) opens this help window. You can navigate topics in the sidebar and use standard scroll and text-selection shortcuts inside each topic.

## Tips

- Hold `Option` while clicking menu items to see hidden power-user variants (where available)
- Sidebar navigation supports arrow keys; press `Return` to open the highlighted project in the detail column
