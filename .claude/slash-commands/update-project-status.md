# /update-project-status

## Description
Creates comprehensive project status updates on Linear for the IDFW unified framework. Posts detailed analysis as new issues or comments with technical metrics and deployment history (Project ID: 4d649a6501f7).

## Tasks
1. Analyze current project state and recent changes
2. Review deployment history and build status
3. Compile technical metrics and test coverage
4. Generate comprehensive markdown report
5. Post to Linear with proper formatting and links
6. Include root cause analysis for resolved issues
7. Document lessons learned and next steps
8. Add progress comments to existing issues
9. Cross-reference Linear IDs with GitHub issues
10. Generate sprint velocity and progress metrics
11. Create actionable recommendations
12. Set up automated status reporting

## Usage Example
```bash
/update-project-status [options]
```

**Options:**
- `--all` : Create full status update with all metrics (default)
- `--issue` : Create as new Linear issue (default)
- `--comment <issue-id>` : Post as comment on existing issue
- `--summary` : Generate status summary without posting
- `--technical` : Include deep technical analysis
- `--metrics` : Focus on project metrics and KPIs
- `--deployments` : Include recent deployment history

## Expected Output
- Comprehensive project state analysis
- Technical metrics and test coverage data
- Deployment history and build status
- Formatted markdown report generated
- Linear issue or comment created
- Root cause analysis included
- Lessons learned documented
- Progress tracking updated
- Cross-references established
- Sprint metrics calculated
- Actionable recommendations provided
- Automated reporting configured

## Success Criteria
- Status update comprehensive and accurate
- Linear integration functional
- Technical metrics current
- Team informed of project state
- Action items clearly defined
- Progress tracking effective
