# /autofix

## Description
Automatically launches priority fix agents concurrently to resolve critical IDFW issues. Runs build error fixes, test failure resolution, and quality monitoring in parallel. Creates comprehensive fix reports and Linear tracking (Project ID: 4d649a6501f7).

## Tasks
1. Scan codebase for critical issues
2. Prioritize issues by deployment impact
3. Launch concurrent fix agents for build errors
4. Deploy test failure resolution agents
5. Run quality monitoring and issue detection
6. Coordinate agent execution and dependencies
7. Implement rate limiting and error handling
8. Generate comprehensive fix reports
9. Create Linear issues for detected problems
10. Verify fixes with automated testing
11. Deploy monitoring agents post-fix
12. Create project status updates

## Usage Example
```bash
/autofix
```

## Expected Output
- Critical issues identified and prioritized
- Multiple fix agents launched concurrently
- Build errors resolved automatically
- Test failures fixed and verified
- Quality issues detected and addressed
- Comprehensive fix report generated
- Linear issues created for tracking
- Automated testing validates fixes
- Monitoring deployed for verification
- Project status updated in Linear

## Success Criteria
- All critical deployment blockers resolved
- Build and tests passing successfully
- Quality metrics improved
- Fix report comprehensive and actionable
- Linear project tracking comprehensive
- Team informed of resolution status
